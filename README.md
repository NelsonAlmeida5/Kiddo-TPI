# Kiddo - TPI

Application web de suivi des devoirs des enfants.

## Stack

- Frontend : Vue.js + Vite
- Backend : AdonisJS
- Base de données : MySQL
- Environnement local : Docker + phpMyAdmin
- Versionnement : Git + GitHub
- CI/CD : GitHub Actions
- Déploiement : Azure

## Structure du projet

```text
backend/   API backend AdonisJS
frontend/  Interface web Vue.js
database/  Scripts SQL setup_database.sql et init_database.sql
docs/      Documentation technique et annexes éventuelles
```
